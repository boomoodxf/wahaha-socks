# Code of Conduct
We are committed to maintaining an open and welcoming environment for all contributors. By participating in this project you agree to abide by the guidelines described here.

- Be respectful and considerate.
- No harassment, discrimination or hate speech.
- Stay constructive in criticism and focus on problems, not people.
- Follow the repository's license and contribution rules.
- If you witness misconduct, report it to the maintainers.

Thank you for helping make this project better.
